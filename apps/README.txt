Breakfast modifications
-----------------------

breakfast: contains Bacon and Toast applications, breakfast utility components,
  testbed tools, command-line tool for Saleae logic analyzer, python
  scripts for interacting with breakfast hardware/deployments.

Blink: script for writing binary to testbed

BaseStation: (obsolete, use breakfast/bacon/BaseStation) fixes for
  handling serial and radio headers of differing lengths.

util/DualUartEcho: obsolete, used for making use of the Telos Mote's
  FTDI chip when we didn't have USB-UART breakout board handy.

Breakfast modifications:

chips: Merge in People Power Co's/Peter Bigot's msp430xv2 code for the x5xx
  series of MCU's and the rf1a core (on the cc430).  Add support for the
  x2xxx series. Support for the on-board sensors on the Bacon mote.
  cross-platform tlvStorage and internal flash support for breakfast
  hardware. Tweaks to stm25p drivers for ROM usage.

lib: CX implementation (plus dead/past implementations)

types: merges for compatibility with PPC code

system: Debug utilities for identifying active tasks with logic
  analyzer.

platforms: definitions for Toast and Bacon platforms.
